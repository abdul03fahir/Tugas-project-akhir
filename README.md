# final-project

Express.js + PostgreSQL starter

## Setup

1. Install dependencies
   ```
   npm install
   ```

2. Create a PostgreSQL database (pgAdmin) and run `sql/create_tables.sql`.

3. Create `.env` based on `.env.example` and set `DATABASE_URL`.

4. Start server
   ```
   npm run dev
   ```

5. Open http://localhost:3000
